Grade Calculator App
📌 Overview
The Grade Calculator App is a Java console application that calculates a student's total marks, average percentage, and final grade based on marks entered for multiple subjects. It also validates user input and handles invalid entries gracefully using exception handling.
✨ Features
Enter student name
Input marks for multiple subjects
Validate marks (0–100)
Handle invalid numeric and non-numeric input
Calculate total marks
Calculate average percentage
Display final grade (A, B, C, D, F)
User-friendly console output
🛠 Technologies Used
Java
Object-Oriented Programming (OOP)
ArrayList Collection
Exception Handling
Scanner Class
